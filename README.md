# 📚 Belajar Irodori – Kuasai Kanji & Kosakata Bahasa Jepang

Selamat datang di **Belajar Irodori**, aplikasi web interaktif untuk mempelajari kanji dan kosakata bahasa Jepang.  
Dibuat berdasarkan data dari buku **Irodori** (level A2, Buku Dasar, dll.) dan dilengkapi berbagai mode latihan yang menyenangkan.

🔗 **Akses langsung:**  
https://fauzired569.github.io/belajar-irodori/Index.html

## ✨ Fitur Utama

- **Latihan Kanji** – Pilih buku dan bab, lalu latih dengan:
  - Flashcard (Grid)
  - Pilihan Ganda (Arti / Bacaan)
  - Menulis Kanji
  - Menulis Bacaan (Hiragana)
  - Mencocokkan Kanji ↔ Arti

- **Latihan Kosakata** – Pilih tingkat (Pemula, Dasar1, Dasar2) dan bab, dengan mode:
  - Flashcard
  - Pilihan Ganda (Arti / Romaji)
  - Menulis Kosakata atau Romaji
  - Mencocokkan Kosakata ↔ Arti

- **Latihan Campuran** – Gabungan soal acak dari Kanji + Kosakata dengan semua mode di atas.

- **Sistem Histori** – Setiap soal yang sudah dilatih akan disimpan di sesi browser. Anda bisa:
  - Mengulang sesi terakhir
  - Memulai latihan baru (hanya soal belum dilatih)
  - Mereset histori untuk mengulang semua soal

- **Tampilan Responsif** – Nyaman digunakan di ponsel, tablet, maupun desktop.

## 🖥️ Demo Singkat

| Mode | Contoh |
|------|--------|
| Flashcard | Klik kartu untuk membalik dan melihat arti/bacaan |
| Pilihan Ganda | Pilih jawaban dari 4 opsi |
| Menulis | Ketik jawaban (kanji, hiragana, atau romaji) |
| Mencocokkan | Pasangkan soal dengan artinya |
